<h1>JavaScript On The Job: An Interactive Scrollytelling Website</h1>

<h2>Description</h2>
JavaScript On The Job is my first attempt at creating an interactive scrollytelling website that aims to help users better understand a complex concept through metaphorical storytelling. Combining HTML, CSS, and JavaScript, this project exemplifies the possibilities of what users can build when learning web development skills. Emphasising interaction, JavaScript On The Job includes dynamic scroll-triggered GSAP animations that are enhanced by Lenis Scroll Smooth for a buttery smooth user experience. Sound interesting? Why not try it for yourself?
<p></p>
<li>https://gavlf.github.io/scrollytelling-JavaScript-on-the-job-gavin/>JavaScript On The Job</li>
<p></p>
Navigating the website is simple: scroll with your middle mouse button and enjoy the story unfolding before your eyes! By the end of your scrolling, you may pick up a thing or two about JavaScript you didn't know before. Whether you are an experienced coder or a complete beginner, JavaScript On The Job is a fun way to experience JavaScript while interacting with an engaging metaphorical story!

<ul>
<li>Light and Dark Mode</li>
